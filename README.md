# TFM_FunestERP

Repositorio para el TFM del Máster en Ingeniería Informática.

## Instalación

- Una vez clonado el repositorio, accedemos a la carpeta e instalamos las dependencias:
    ~~~~
    cd TFM_FunestERP
    npm install
    ~~~~
- Lanzamos los tests:
    ~~~
    npm test
    ~~~~
- Lanzamos la aplicación:
    ~~~
    npm start
    ~~~~
